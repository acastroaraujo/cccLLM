# Package index

## LLM Datasets

- [`rulings`](https://acastroaraujo.github.io/cccLLM/reference/rulings.md)
  : Data set of rulings
- [`person`](https://acastroaraujo.github.io/cccLLM/reference/person.md)
  : Data set of individual judges participating in the rulings
- [`amicus`](https://acastroaraujo.github.io/cccLLM/reference/amicus.md)
  : Data set of amicus interventions in each ruling
- [`articles`](https://acastroaraujo.github.io/cccLLM/reference/articles.md)
  : Edge list of rulings and articles of the Colombian Constitution
- [`rj_citations`](https://acastroaraujo.github.io/cccLLM/reference/rj_citations.md)
  : Edge list of citations to binding precedent ("cosa juzgada")

- [`reviews`](https://acastroaraujo.github.io/cccLLM/reference/reviews.md)
